About me

I’m Amy Wolgemuth

I am a Full-Stack Web Developer

After graduating from highschool in 1988, I went to college at Bowling Green State University in Bowling Green, Ohio. I received a Bachelor of Science in Computer Science, 1992. Rather than embarking on a programming career, I chose to be a stay-at-home mom to my 5 children. When my youngest was old enough for pre-school, in 2005, I started working at IDEXX Laboratories located in Eau Claire, Wisconsin. IDEXX created, maintained and supported the major veterinary clinic software in the United States. I was so glad to have found my niche!

I started as a Software Support Consultant I and moved up the ranks to Software Support Consultant III/Resource Technician by 2012. This was a high profile and demanding position, but I loved it! After being an RT for 5 yrs, in 2017, through a turn of events, I was able to move to Nashville, Tennessee with 3 of my children.

I began working for Trinisys, LLC in Brentwood, Tennessee in June, 2018 and still work there. Trinisys is innovative and passionate about helping our clients solve their buisiness problems through software solutions. I am Software Support Analyst and work with our Clients to resolve software issues and errors. I wanted to grow far along with the innovative and trustworthy company so I enrolled in Full-stack Coding Bootcamp at Vanderbilt University and am on my way to becoming a software developer!

React Clicky Game

Memory game built with React.

For this assignment, I created a memory game with React. This assignment required me to break up my application's UI into components, manage component state, and respond to user events.

Created a new React application using Create React App. The application renders different images to the screen. Each image listens for click events. It keeps track of the user's score. The score will be incremented when clicking an image for the first time. The user's score resets to 0 if an image is clikced more than once. Every time an image is clicked, the images on the page get shuffled in a random order. After an incorrect guess, the user's score is reset and the game restarts.
